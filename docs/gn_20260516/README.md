# 2026-05-16 纳指/黄金 ETF 动量测试记录

## 结论摘要

本轮测试的核心结论：

1. A股 `159941/518880` 和美股 `QQQ/GLD` 在同主题下仍会有不同结果，主要来自交易时段、人民币计价、汇率、QDII 折溢价和 ETF 自身价格序列差异。
2. 美股 `QQQ/GLD` 两资产策略在当前选定口径下更适合高切换阈值，粗扫显示 `switch_buffer=2.5` 最优或处在最优平台区。
3. A股 `159941/518880` 若使用原 1.1 参数骨架并去掉半仓、过热、分数限制，则不偏好高阈值；`1.00~1.05` 更合适。
4. “分数是否限制在 0 到 5”不是美股高阈值的主因。加上 `0<score<5` 后，美股两资产仍偏好较高阈值。
5. 半仓等阴线规则对美股 `QQQ/GLD` 全样本有拖累，对近 3 年/1 年有帮助；不适合作为主版本默认增强项。
6. A股 `159941/518880` 在原 1.1 骨架下，去掉半仓后收益明显提高。

## 原始动量基线

只测纳指/黄金两个资产，原始动量，不加额外条件：

| 组合 | 年化 | MaxDD | Sharpe |
|---|---:|---:|---:|
| A股 `159941/518880` | 22.15% | -28.89% | 1.116 |
| 美股 `QQQ/GLD` | 约 21.3% | -32.87% | 约 1.18 |

结论：裸动量下两边差距不大，后续差异主要来自条件叠加和价格序列。

## 美股 R2 扫描

口径：`QQQ/GLD`，原始 25 日加权 log slope，先不加目标波动。

| R2 阈值 | 年化 | MaxDD | Sharpe |
|---:|---:|---:|---:|
| 无 R2 | 21.27% | -32.87% | 1.181 |
| 0.20 | 23.71% | -25.12% | 1.242 |
| 0.30 | 22.85% | -16.74% | 1.228 |

结论：`R²=0.30` 更防守，回撤明显小，作为后续美股两资产参数。

## 美股目标波动

口径：`QQQ/GLD`，`R²=0.30`。
